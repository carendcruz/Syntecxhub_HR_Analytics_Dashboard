# Employee Attrition & HR Analytics Dashboard

## 📌 Project Overview
This project focuses on analyzing employee data to identify patterns related to attrition, job satisfaction, salary distribution, and workforce demographics.

The dashboard was created to help organizations make data-driven HR decisions by understanding workforce trends and employee retention factors.

---

## 🎯 Objectives
- Analyze employee attrition trends
- Understand workforce demographics
- Identify factors affecting employee satisfaction
- Compare salary and department-wise performance
- Create an interactive HR analytics dashboard

---

## 🛠️ Tools & Technologies Used
- Power BI
- Microsoft Excel
- Data Cleaning
- Data Visualization

---

## 📊 Dashboard Features
- Total Employees Overview
- Attrition Rate Analysis
- Department-wise Employee Distribution
- Gender Distribution
- Job Satisfaction Analysis
- Salary Insights
- Interactive Filters & Slicers

---

## 📈 Key Insights
- Some departments experienced higher attrition rates than others.
- Lower job satisfaction was linked to increased employee attrition.
- Salary distribution varied across departments and job roles.
- Workforce demographic analysis provided insights into employee diversity.

---
